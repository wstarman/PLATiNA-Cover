# PLATiNA Cover

### 簡介

這是一個可以在電腦上產生多個可穿透點擊的半透明視窗的程式，視窗內可以填滿純色或者放置一張圖片。

最初是作為 PLATiNA::LAB 背景亮度過高的簡易調整措施，並且參考了在行動裝置上功能相近的 "Deemo Cover" 命名。

除了調整亮度外，亦可以用於上/下隱、放置筆記，或者當作射擊遊戲的準心等。

<img src=".\readme_assets\demo.png" alt="demo" style="zoom:33%;" />

### 使用方法與注意事項
- 可以將控制視窗最小化
- 將存檔命名為"default"即可於程式啟動時自動載入配置。
- 無法覆蓋「全螢幕」模式的程式上，但可以覆蓋「無邊框視窗」模式的程式。
- 若不慎覆蓋了整個螢幕而且還把控制視窗最小化，可以按 Alt + Tab 把控制視窗叫回來。



# PLATiNA Cover

### Introduction

This is a program that creates multiple semi-transparent windows on a computer that can be clicked through. The content of each window can either be filled with a solid color or display an image.

It was originally developed as a simple brightness adjustment method for the background of PLATiNA::LAB, and its name was inspired by a similar feature on mobile devices called “Deemo Cover”.

In addition to brightness adjustment, it can also be used for showing/hiding overlays, placing notes, or functioning as a crosshair in shooting games.

### Usage and Notes

- You can minimize the control window.
- Saving a configuration file with the name “default” will allow it to be automatically loaded when the program starts.
- It cannot overlay applications running in full-screen mode, but it can overlay borderless windowed applications.
- If the entire screen is accidentally covered and the control window is minimized, press Alt + Tab to bring the control window back.
